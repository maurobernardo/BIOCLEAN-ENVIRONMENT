# Imagens para o site

Para que todas as imagens carreguem corretamente no Vercel, coloque os ficheiros nas pastas indicadas:

## `public/images/`

| Ficheiro | Uso |
|----------|-----|
| `logo.png` | Logo no Navbar e Footer |
| `ceo.png` | Foto do CEO (secção Sobre Nós) |
| `Fundo.JPG` | Imagem de fundo do Hero (atenção: extensão .JPG) |
| `og-share.png` | **Preview ao partilhar no WhatsApp** – tamanho recomendado 1200×630 px |

## `public/galeria/`

| Ficheiro | Uso |
|----------|-----|
| `Nampula.jpeg` | Galeria de projetos |
| `Nampula1.jpeg` | Galeria de projetos |
| `HST.jpeg` | Galeria de projetos |
| `Nacala.jpeg` | Galeria de projetos |
| `Zambezia.jpeg` | Galeria de projetos |
| `Social.jpeg` | Galeria de projetos |

**Importante:** A pasta `public/` é servida diretamente. O Vercel é sensível a maiúsculas/minúsculas – use exatamente os nomes indicados.
